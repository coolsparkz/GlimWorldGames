# GlimWorldGames
Jump into GlimWorld — a fast, expanding library of games, worlds, and interactive adventures.

GlimWorldGames is an evolving collection of bgames, creative worlds, and interactive experiences designed to grow over time. Whether you're exploring new environments, trying out experimental mechanics, or jumping between different mini‑games, GlimWorldGames brings everything together in one place.

Our goal is to build a fun, fast, and accessible hub where players can discover new adventures and creators can showcase ideas. As the project expands, expect more worlds, more gameplay styles, and more surprises.

You can start playing right now:

- **Main GlimWorld experience:** https://glim-world--lomov0.replit.app  
- **GlimWorldGames library:** https://coolsparkz.github.io/GlimWorldGames/
- **Other Info:** https://github.com/coolsparkz/GlimWorld
- And more coming soon!

## 🛠️ Contributing
GlimWorldGames is an expanding project, and contributions are welcome as the library grows.

If you’d like to help improve the games, add new worlds, fix bugs, or enhance the website, here’s how to get started:

1. **Fork** the repository  
2. **Create a new branch** for your feature or fix  
3. **Make clean, well‑documented changes**  
4. **Submit a pull request** with a clear description of what you added or improved  

If you're unsure where to begin, feel free to open an issue and discuss ideas before contributing.

## 🔐 Security
We take the security of GlimWorldGames seriously. If you discover a vulnerability, please report it responsibly so we can address it quickly.

**How to report security issues:**
- Email: **lomov0@eq.edu.au** or **github.com**
- Preferred language: **English**
- Please **do not publicly disclose** vulnerabilities before they are fixed
